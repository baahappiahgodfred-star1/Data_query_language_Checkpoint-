# DQL Checkpoint - SQL

## What I did
In this checkpoint, I wrote SQL queries (DQL) to retrieve data from the database.

The database contains 3 tables:
- Customer
- Product
- Orders

---

## Queries implemented

1. Display all customers
2. Get products with price between 5000 and 10000
3. Sort products by price (descending)
4. Calculate total orders, average, max and min amount
5. Count orders per product
6. Find customers with more than 2 orders
7. Count orders per month in 2020
8. Show product name, customer name and order date
9. Get orders made 3 months ago
10. Find customers who never ordered

---

## Conclusion
This checkpoint helped me practice SQL queries like SELECT, WHERE, GROUP BY, HAVING and JOIN.
